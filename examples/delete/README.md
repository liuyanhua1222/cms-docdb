# delete — 使用说明

## 什么时候使用

- 用户说"帮我把 xxx 文件删了"、"删除这个文件"

## ⚠️ 危险操作确认流程

删除文件是**高风险操作**，必须按以下流程执行：

1. 鉴权预检（按 `cms-auth-skills/SKILL.md` 获取 appKey）
2. **先向用户确认**："确认要删除文件 [文件名] 吗？此操作[可/不可]撤销。"
3. 用户明确确认后，调用 `scripts/delete/delete-file.py`
4. 返回删除结果

## 删除模式

| 模式 | 参数 | 说明 |
|---|---|---|
| **逻辑删除（默认）** | 不加 `--physical` | 文件移入回收站，可从回收站恢复 |
| **物理彻底删除** | 加 `--physical` | 从回收站彻底抹除，**不可恢复** |

## 用户话术示例

- "帮我把这份文档删了"
- "删除周报 xxx"
- "彻底删除这个文件"
