# manage — 使用说明

## 什么时候使用

- 用户说"帮我把 xxx 文件重命名"、"把这个文件改个名字"
- 用户说"帮我把 xxx 文件移到 yyy 文件夹"
- 用户说"把这个文件移走并改个名"

## 标准流程

1. 鉴权预检
2. 获取目标文件的 fileId（如不知道，先搜索或浏览找到）
3. 调用 `scripts/manage/update-file-property.py`
4. 返回操作结果

## 冲突处理

同名冲突时有三种策略：

| 策略 | 参数 | 说明 |
|---|---|---|
| 静默覆盖 | `--cover` | 直接覆盖已有文件 |
| 自动重命名 | `--auto-rename` | 自动追加数字后缀，如 `文件名(1).pdf` |
| 报错 | 二者都不传 | 后端报错，Agent 需处理 |

## 用户话术示例

- "帮我把这份文档改个名"
- "把这个文件移到 AI 生成文件夹"
- "文件名重复了，帮我自动重命名"
